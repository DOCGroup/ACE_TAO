ACE Tutorial 011
Passing non-trivial data through an ACE_Message_Queue

So, this time we stuffed an object into the message queue instead of just text data. Each time required two object constructions (and subsequent destructions) and two "deep" copy operations on the object. There might actually be times when this is OK for your application but I prefer to keep those things down to a minimum. In the next tutorial I'll show you a way to do that.

ACE+TAO Documentation

Documentation is separated by components:

  • The ACE library

  • ACE SSL Adapters

  • ACE Reliable Multicast components

  • The TAO library and components

  • TAO_PortableServer: TAO's Portable Object Adapter

  • TAO_DynamicAny: TAO's implementation of Dynamic Anys.

  • TAO_IORManipulation: TAO extension to manipulate IORs.

  • TAO_IORTable: TAO's ad-hoc adapter for simple Object Keys.

  • Implementation Repository: TAO's object activation daemon.

  • TAO_ESF: TAO's Event Service Framework, a collection of template classes used to implement TAO Event Services.

  • TAO_RTEvent: TAO's RT Event Service implementation.

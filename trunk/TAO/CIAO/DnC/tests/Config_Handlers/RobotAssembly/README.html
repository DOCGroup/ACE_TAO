This document describes how to compile/run the Robot Assembly with the new DnC implementation.

Installing

  • Make sure you have the latest DnC code by running [in ACE_wrappers/TAO/CIAO/DnC directory] cvs up
  • Get the latest cidl compiler (if you did not compile from source) from (precompiled binaries for linux, windows, darwin, and solaris are provided): http://www.dre.vanderbilt.edu/cidlc/binary/ Copy the file into the ACE_wrappers/TAO/CIAO/bin directory.
  • Generate the make/project files for the DnC directory by using ACE_wrappers/bin/mwc.pl script:
    • linux: [in $CIAO_ROOT/DnC] mwc.pl
    • windows: [in %CIAO_ROOT%/DnC] mwc.pl
  • Compile the DnC code with the generated solution/make files.
  • Generate the project/make files for the RobotAssembly directory: [in RobotAssembly root folder] mwc.pl
  • Compile the generated solution/make files

Running the test

  • Create a new directory [in RobotAssembly root folder] test (or any other name you want) and change the working directory to that.
  • Download the files into the newly created directory:
    • http://www.dre.vanderbilt.edu/~turkaye/dnc/RobotAssembly/NodeDaemon.txt [and rename that file to NodeDaemon.pl]
    • http://www.dre.vanderbilt.edu/~turkaye/dnc/RobotAssembly/deployment.dat
  • Start daemons by running NodeDaemon.pl
  • Start the execution manager: ACE_wrappers/TAO/CIAO/DnC/ExecutionManager/Execution_Manager -o EM.ior -i deployment.dat
  • Start the executor: Note: You should give the correct path of your ACE_wrappers directory.
    ACE_wrappers/TAO/CIAO/DnC/RepositoryManager/executor -p /ACE_wrappers/TAO/CIAO/DnC/tests/Config_Handlers/RobotAssembly/package.tpd -d /ACE_wrappers/TAO/CIAO/DnC/tests/Config_Handlers/RobotAssembly/DeploymentPlan.cdp -k file://EM.ior

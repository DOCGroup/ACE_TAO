ACE+TAO+CIAO Documentation

Documentation is separated by components:

  • The ACE library

  • The ACEXML library

  • ACE SSL Adapters

  • ACE Reliable Multicast components

  • ACE Components for QoS-enabled networking APIs.

  • The TAO library and components

  • TAO_PortableServer: TAO's Portable Object Adapter

  • TAO_DynamicAny: TAO's implementation of Dynamic Anys.

  • TAO_DynamicInterface: TAO's implementation of DII/DSI.

  • TAO_SmartProxies: TAO's smart proxies support library.

  • TAO_IORManipulation: TAO extension to manipulate IORs.

  • TAO_IORTable: TAO's ad-hoc adapter for simple Object Keys.

  • TAO_Strategies: Additional configuration options and protocols for TAO.

  • TAO_RTPortableServer: The Portable Object Adapter with RT-CORBA support

  • TAO_RTCORBA: Additional components to support RT-CORBA

  • TAO_PI: TAO Client Portable Interceptor support

  • TAO_PI_Server: TAO Server Portable Interceptor support

  • TAO_AnyTypeCode: TAO AnyTypeCode support

  • Implementation Repository: TAO's object activation daemon.

  • TAO_ESF: TAO's Event Service Framework, a collection of template classes used to implement TAO Event Services.

  • TAO_RTEvent: TAO's RT Event Service implementation.

  • TAO_CosEvent: TAO's COS Event Service implementation.

  • TAO_CosNaming: TAO's COS Naming Service implementation.

  • TAO_CosNotification: TAO's COS Notification Service implementation.

  • TAO_CosTime: TAO's COS Time Service implementation.

  • TAO_CosTrader: TAO's COS Trader Service implementation.

  • TAO_AV: TAO's Audio/Video Streaming Service implementation.

  • TAO_Security: TAO's CORBA Security Service implementation.

  • TAO_SSLIOP: TAO's SSLIOP protocol implementation.

  • TAO_PortableGroup: TAO's PortableGroup implementation.

  • TAO_PSS: TAO's Persistent State Service implementation.

  • TAO_IFR: TAO's Interface Repository Service implementation.

  • CIAO: CIAO's core implementation.

  • DAnCE: CIAO's Deployment and Configuration Engine.

  • Config Handlers: CIAO's Config Handlers.

CIAO - Component-Integrated ACE ORB

Introduction

CIAO is TAO's implementation of CORBA Component Model (CCM) specially optimized for distributed real-time embedded (DRE) systems. This page contains links to other documents and resources for using CIAO.

Documents

Here is a set of UML diagrams that we use to figure out the relationship among all the different classes in CCM. You can browse the related part from the following list:
  • Basic component interfaces
  • Home and configuration interfaces
  • Basic container interfaces
  • Extended container interfaces
  • Deployment and installation interfaces
All these diagrams are generating from the CCM.vsd Visio file (The same file in Visio V.5 format is here.)
